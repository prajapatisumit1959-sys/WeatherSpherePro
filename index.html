About the project

Why I built a weather console instead of a weather “app.”

Most student weather widgets stop at a temperature and an icon. WeatherSphere Pro is built like an actual operations console — gauges for UV and wind direction, a live AQI dial, hourly and 7-day projections, and a side-by-side city comparison mode — to demonstrate real dashboard engineering, not just an API call wrapped in a card.

Every metric on this page — humidity, pressure, visibility, rain probability, sunrise and sunset — comes from a single live data source, normalized and rendered with vanilla JavaScript and Chart.js. No frameworks, no build step: just HTML, CSS and JS doing the work.
