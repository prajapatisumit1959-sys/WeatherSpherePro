# WeatherSphere Pro

WeatherSphere Pro is a modern weather application built using HTML, CSS, and JavaScript. It provides real-time weather information, air quality insights, and a clean responsive interface for users to check weather conditions from anywhere.

# Live Demo

🔗 https://6a38c186cee129e58a613736--weather-sphere-pro.netlify.app/

# Features

* Real-time weather updates
* Search weather by city name
* Air Quality Index (AQI) information
* Responsive design for desktop and mobile devices
* Clean and modern user interface
* Fast and lightweight performance

# Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Weather API
* Netlify (Deployment)
